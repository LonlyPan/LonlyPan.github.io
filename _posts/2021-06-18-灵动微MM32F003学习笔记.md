## 破解 Keil MDK
>注意！注册机文件会被杀毒软件识别为危险程序！当然，任何注册机都会被杀毒软件识别为危险程序！请自行评估风险后再决定使用！如果你决定使用注册机进行注册，请先将杀毒软件的实时防护功能暂时关闭，或者将注册机程序文件设置为信任白名单中

1. 从keil安装包中找到【KeilMDK5Keygen.exe】这个文件，运行它，出现如下界面
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624066444943.png)
2. 以【管理员】方式运行Keil µVision 5，然后通过菜单栏“File->License Management…”打开【License Management】对话框。
![enter description here](https://markdown.xiaoshujiang.com/img/spinner.gif "[[[1624066485778]]]" )
3. 将上图Keil MDK License Management里显示的CID值填入破解器的CID栏，Target下拉选择ARM。点击【Generate】按钮生成Key
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624066561793.png)
4. 切到License Management 对话框，将刚才生成的key信息输入到【New License ID Code】输入框中，然后点击旁边的【Add LIC】即可。如果你启动Keil µVision 5是没有以管理员权限运行，会提示失败，重新以管理员权限打开Keil µVision 5即可。这版注册机可用到2032年。
