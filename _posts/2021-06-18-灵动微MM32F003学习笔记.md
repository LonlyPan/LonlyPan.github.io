# 开发环境搭建
先去官网[MM32F00系列文档](http://www.mindmotion.com.cn/download.aspx?cid=2576)下载以下五份资料:
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624065546997.png)
## MDK keil安装

**百度网盘下载地址：**
 - 链接：https://pan.baidu.com/s/148udLx1hpLYcxpXEysjmxA 
 - 提取码：x47j 
 - 
1. 自定义安装路径：自行修改路径即可，不过要注意：安装路径一定不要包含中文名！然后在设置一些简单的信息（名字、公司、邮箱等，这些信息可以不填，直接输入空格也可）就开始安装了，如图 1.2 所示等待安装完成：
 ![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624008805635.png)
2. 客户信息这里随便填就好了 别太认真 ！
 ![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624064467459.png)
3. 提示安装ULINK驱动，选择安装
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624064536351.png)
4. 最后点击 Finish 即可完成安装，随后， MDK 会自动弹出 Pack Installer 界面，如图所示：
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624064597572.png)
5. 	选择安装包程序就会自动去 KEIL 的官网下载各种支持包，不过这个过程有可能失败，如图所示：
![enter description here](https://LonlyPan.github.io/images/Posts/2021-06-18-灵动微MM32F003学习笔记/1624064699414.png)
遇到这种情况，我们直接确定，然后关闭这个包安装器即可。支持包，我们可以自行去官网下载，
下载地址： http://www.mindmotion.com.cn/download.aspx 。 注册登录后，在 MDK pack 文件下载。 这
里我们以： MM32F103 开发为例，则至少需要安装： CMSIS 和 MM32F103 的器件支持包， CMSIS 包
MDK5.18 自带了，所以不需要单独再安装
