# 下载安装

FreeCAD官网：https://www.freecadweb.org/

支持windows、mac、linux。下载所需版本，安装全部保持默认选项，可以更改安装路径。

## 更改语言

1. 软件左上角选择 `Edit`->`Preferences`；
2. 然后在弹窗界面选择 `General`, 
3. 在`Language`一栏中的 `Change Language` 下拉选项框选 择**简体中文**。
4. 最后单击 `Apply` 生效配置，点击 `OK`  退出完成。
![图 1](../images/Posts/2020-06-23-FreeCAD%E5%AD%A6%E4%B9%A0%E7%AC%94%E8%AE%B0/freecad%20language.png)  

## 更改鼠标操作
软件左上角选择 `Edit`->`Preferences` -> Display -> 导航栏（Navifation）

3d视图界面，右下角有操作提示（视图，勾选状态栏显示）

## 更改启动界面
软件左上角选择 `Edit`->`Preferences` 选择启动界面的工作台

## 背景颜色
软件左上角选择 `Edit`->`Preferences` -> Display -> Cilors

## 零件设计 bool 检查
软件左上角选择 `Edit`->`Preferences` -> 零件设计 -> 常规 ：全部勾选

# 操作技巧

选中物体，空格建控制其可见/不可见

选中的呈现绿色，鼠标预选黄色

模型消失（不在屏幕区域内），左上 放大镜图标

右上魔方，切换不同视角

组合浏览器，选中图标，可以在下方得到属性栏快速修改模型属性。

完全约束的草图为绿色

## 齿轮

两个齿轮啮合间距 = （模数 x （齿数1 + 齿数2）） / 2。旋转角度 = 360/齿数/2
## 编辑STL

1. 导入模型
2. 工作台选择 Part
3. Part - Creat shapefrom mesh 转为网格，精度0.01
4. Part - Convert to solid  转为实体
5. Part - Refine shape 平滑特征
6. 工作台转为 part design，新建 Body 

3-5每部操作后都可以删除上一步模型。

## 阵列

选择阵列时，特征如果是依附与某个实体（圆角，凹坑基于立方体），则应先阵列该实体（立方体），否则特征就无法阵列（无实体可依附，即悬空，软件判断为错误，无法阵列）

有些圆角无法阵列，就很离谱。好像只有拉伸和切割可以阵列
