## 阵列

选择阵列时，特征如果是依附与某个实体（圆角，凹坑基于立方体），则应先阵列该实体（立方体），否则特征就无法阵列（无实体可依附，即悬空，软件判断为错误，无法阵列）

有些圆角无法阵列，就很离谱。好像只有拉伸和切割可以阵列

### Kicad StepUp

**Installation**
KicadStepUp is part of the external workbenches, and can be automatically installed using the **AddonManager** Add-on Manager which comes bundled with FreeCAD 0.17, under the Tools → Add-On Manager menu
