层级不超过三级
命名规则：xx_file_name  (x代表数字，两位数递增，下同；全小写，下划线连接单词)

<!--more-->
## 软件项目

```
00_Priority README   /* 存放技术文档 \*/
01_source            /* 存放软件源码和已迭代源码 \*/
  |____01_new_source /* 存放最新源码 \*/
     |____01_新源码名Vx.x
     |____ ...       /* 有的源码可能有多种输出 \*/
  |____02_old_source /* 存放已迭代源码 \*/
     |____01_旧源码名Vx.x
     |____ ...
02_tools             /* 相关软件及使用文档\*/
  |____01_xx软件     /* 每个软件单独一个文件夹 \*/
  |____...
03_documents         /* 软件设计参考文档 \*/
04_others   		 /* 软件相关其它文档 \*/
README.txt           /* 建议每个文件夹下都有该文件，所处文件夹的阅读索引及必要说明 \*/
```

## 软硬件项目

若是软硬件项目，将技术文档提取到根目录，其它保持一致

```
00_Priority README   /* 存放技术文档 \*/
01_Firmware   /* 软件项目 \*/
  |____01_source     /* 存放硬件源码和已迭代源码 \*/
  |____02_tools      /* 相关软件及使用文档\*/
  |____03_documents  /* 软件设计参考文档 \*/
  |____04_others   	 /* 软件相关其它文档 \*/
02_Electronics/* 硬件项目 \*/
  |____01_source     /* 存放硬件源码和已迭代源码 \*/
  |____02_tools      /* 相关软件及使用文档\*/
  |____03_documents  /* 硬件设计参考文档 \*/
  |____04_others   	 /* 硬件相关其它文档 \*/
03_others     /* 项目相关其它文档 \*/
README.txt  		 /* 建议每个文件夹下都有该文件，所处文件夹的阅读索引及必要说明 \*/
```
