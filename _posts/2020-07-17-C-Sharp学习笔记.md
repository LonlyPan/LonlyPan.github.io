# 第一季：基础语法

## WHY？

计算机系统 = 应用软件 + 操作系统 + 硬件

应用软件是使用软件开发代码一行行编写出来的（包括界面）。

**为什么学习软件开发**
- 软件开发工资高，职位多
- 软件开发是创造性的工作，不依附于厂商，受外界影响较小
- 软件开发的发展前景大
- 软件开发见效快，对硬件要求低
 
 **为什么学习C#*
 - C#拥有丰富的类库，对于初学者较简单
 - C# 完全面向对象的、类型安全的语言
 - 程序开发的逻辑和面向对象的概念都是一样的，各语言通用
 
 **后期发展方向**
  - 桌面应用程序 - Windows Form
  - Internet 应用程序 - ASP.NET
  - 手机应用程序 （目前已逐步淘汰）
  - Unity3D 游戏开发或者虚拟现实
  - 微软产品二次开发：活动目录，SharePoint，Dynamics CRM， Dynamics AX
   
   ## 开发环境搭建
   
   **开发平台的基础要求**
   - 操作系统（Win10专业版以上，不要Home版）
   - 数据库平台：SQL Server 2008 R2/2012/2014/2016
   - 软件平台：Visual Studio 2017
   
推荐开发架构：Win10 + SQL Server 2012 + Visual Studio 2015

### 写代码基本规范

- 代码中所有的标点都是英文半角
- 每一行语句以分行结束
- 所有代码书写大小写敏感
- 可通过F5来运行所写的程序
- 可通过生成解决方案来帮助我们来排查语法错误

### 基本设置

- 设置行号：工具 选择 文本编辑器 C# 行号 勾选
- 设置字体：Console
- 设置代码自动保存
- 设置启动项 解决方案 属性页 通用属性 启动设置
- 项目的移除和卸载，右键项目卸载，卸载为了不占用CPU资源，移除至只从文件树中移除不显示，并非真正删除源文件。
